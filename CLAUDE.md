## Project summary
- **PharmaSynth** — a VR chemistry-lab education game for **Meta Quest 3**, built in **Unity 6000.5.2f1** (URP, OpenXR + XRI 3.5.1). Client handoff from a discontinued capstone team — an **audit-and-continue** build, not from-scratch. **Hard deadline: August 31, 2026** (contract).
- **Game concept:** first-person guided lab sim, "PharmaSynth: Gear Up, Synth It Up!". **11 experiments** (Tutorial: Methane; Prelim: Chemical Compounding, Ethyl Alcohol; Midterm: Benzoic Acid, Acetanilide, Acetone, Chloroform; Final: Benzamide, Aspirin, Caffeine, Wine Making). Per-experiment loop: intro cutscene → reagent prep → synthesis → chemical tests → data sheet/quiz → end cutscene (ALWAYS, success or fail variant) → grade screen. Bayesian Knowledge Tracing mastery + rubric grade, **two-part 90% gate** to advance. NPCs: **Pharmee** (robot guide, subtitles + beeps) + **Dr. Jimenez** (examiner, no hints; AI-generated rigged model with Idle/Talk/Walk animator since 2026-07-10 — drive the "Talking" bool).
- **Master plan:** `C:\Users\MSI\.claude\plans\you-are-the-best-cozy-possum.md` (design spec, architecture, roadmap; **§10 = the confirmed client workflow**). Update it when decisions change.
- **Chemistry authority = manuscript Appendix C** (never the storyboard). Client flags — all VERIFIED against the extracted manuscript 2026-07-10, evidence in `Docs/manuscript-reconciliation.md`: Benzoic Acid = benzaldehyde + KMnO₄ (Appendix C Exp 4 procedure is a verbatim copy-paste of the Ethyl Alcohol fermentation — confirmed defect; materials list names Benzaldehyde), Acetanilide acylating agent = acetyl chloride per Appendix C (intro prose's "anhydride" is wrong), Benzamide nitrous test = sodium nitrite (reagents-header "nitrate" is a typo; procedure body says nitrite). **Manuscript covers 8 of the 11 modules — Methane/Aspirin/Caffeine are game-authored (not in it).** The manuscript PDF (`Docs/Documentations/manuscript.pdf`) extracts to text via `pdftotext -layout` (Read tool can't open PDFs here).

## Known gotchas (carry forward)
- `Unity_RunCommand` compiles inside `Unity.AI.*` → fully-qualify/alias `UnityEngine.UI.*` (`UImage`, `UButton`). `System.Reflection`/`ISet`/`GetInstanceID` blocked or error. File writes + `AssetDatabase.DeleteAsset` flagged "requires user interaction" → use Bash for files, load-or-overwrite for assets. **Menu items are exempt** (that's why DevCapture works).
- **Never RunCommand while the user is in Play mode** (`OpenScene` throws; scene edits force-exit play) — check `Unity_ManageEditor GetState` first. One scene per command; `AssetDatabase.Refresh()` before cross-scene asset loads; a `Refresh:true` menu execute can swallow the run in the domain reload (refresh first, then run).
- Edit mode: `OnEnable`/`Awake` don't fire on `AddComponent` → every component needs a public `Bind()`/`SetX()` seam (pattern used throughout). Never `renderer.material` in edit mode (use `sharedMaterial`/MaterialPropertyBlock). `AddComponent<LiquidPhysics>` needs a Renderer host (`CreatePrimitive(Cube)`).
- Before deleting a script, grep ALL of `Assets/` for the type name. FBX imports have no colliders. The ChemLab `_WithLiquid` prefabs had glass/liquid roles reversed (verify mesh names, not GO names); the pack's Glass shader is not URP (materials already ported).
- MCP "named pipe not found" = Unity busy/reloading → wait for `Logs/Editor.log` quiet, retry. MCP "Connection revoked" = AI-seat licensing hiccup (Editor.log shows `Licensing::Client` 404s); it can self-heal after a reload, else the user re-approves under Project Settings → AI → Unity MCP — meanwhile use the `SelfTestAutoRun` request-file fallback. `Unity_Camera_Capture` fails on every camera here → use DevCapture.
- Transparent geometry (curtains/glass) doesn't z-write → text/canvas visibility is decided by **sortingOrder** (values in the UI section above).
- The suite's `[ExperimentLauncher] Unknown moduleId: does-not-exist` warning is a deliberate negative test, not a defect.
- Poppler/PDF: TEMP is hijacked → the Read tool cannot open PDFs; use `"C:/Program Files/Git/mingw64/bin/pdftotext.exe"` or python pypdf. The manuscript exports via gdoc `export?format=txt`.
- Internet works via `curl` (kenney.nl fine; github-raw 429). CC0 sources: Kenney, OpenGameArt, Poly Haven/ambientCG, Quaternius.
